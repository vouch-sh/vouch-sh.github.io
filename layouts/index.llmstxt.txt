# Vouch

> Vouch is an open-source credential broker that replaces long-lived secrets (AWS keys, SSH keys, GitHub PATs, registry tokens) with short-lived credentials after FIDO2 hardware verification. One YubiKey tap, up to 8 hours of access, full audit trail.

Vouch eliminates static secrets across your development workflow. It federates identity through OIDC, so every credential is tied to a verified hardware key and expires automatically. No more key rotation scripts, no more offboarding checklists, no more secrets in config files.

Every HTML page on vouch.sh is also served as raw Markdown at the same URL (e.g. `/docs/getting-started/index.md`). Token counts below are approximate (chars ÷ 4).
